Cuestionario

Complete las siguientes oraciones.

  1. El elemento______inserta una regla horizontal.

    hr.

  2. Para marcar un superíndice se utiliza el elemento______ y para marcar un subíndice se utiliza el elemento______.

    sup, sub.

  3. El elemento de encabezado menos importante es______ y el elemento de encabezado más importante es______.

    h6, h1.

  4. El elemento______marca una lista desordenada.

    ul.

  5. El elemento ______marca un párrafo.

    p.

  6. El atributo______ en un elemento input inserta un botón que, cuando se presiona, restablece el contenido del formulario.

    type = "reset".

  7. El elemento______marca una fila de una tabla.

    tr.

  8. Los______se usan por lo general como caracteres de enmascarado de un cuadro de contraseña.

    Asteriscos.
